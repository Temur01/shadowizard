# What is this?

Perfect designing package

# Installation

`npm i myshadow -- save`

Then...

```
import {myshadow} from 'myshadow'

myshadow({
    shadow_type:'soft',
    padding:false
});


```

## Options

myshadow supports 2 options, both of which are optional:

- _shadow_type_ - _hard/soft_ (Defaults to soft)
- _padding_ - _boolean_ (Defaults to false)
